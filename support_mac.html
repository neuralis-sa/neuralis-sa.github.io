Support macOS - Guide d'installation ISL

Suivez les étapes ci-dessous pour installer et autoriser ISL Light Client sur votre Mac, permettant à notre équipe technique de vous assister à distance.

1. Vérifier votre version de macOS

Avant de commencer, identifions la version de votre système d'exploitation.

Cliquez sur le logo Apple en haut à gauche de votre écran, puis sélectionnez "À propos de ce Mac".

Une fenêtre s'affichera indiquant votre version de macOS (par exemple : macOS Monterey, Ventura, Sonoma, etc.). Prenez note de cette version pour référence.

2. Télécharger ISL Light Client

Obtenez le logiciel nécessaire pour la session de télémaintenance.

Télécharger ISL Light Client

Étape 1 : Un code de session vous sera demandé. Un de nos collaborateurs vous le fournira lors de votre demande de support.

Étape 2 : Entrez le code fourni, puis cliquez sur "Rejoindre".

Étape 3 : Le fichier "ISL Light Client" sera téléchargé automatiquement. L'icône de téléchargement apparaîtra dans la barre de raccourcis de votre navigateur.

Cliquez sur l'icône de téléchargement pour afficher les fichiers récemment téléchargés.

3. Lancer ISL Light Client

Exécutez le logiciel et rejoignez la session de support.

Étape 1 : Cliquez sur le fichier "ISL Light Client" téléchargé pour l'exécuter.

Étape 2 : Une fenêtre de sécurité s'affichera. Cliquez sur "Ouvrir" pour confirmer.

Étape 3 : Le logiciel ISL Light Client se lancera et rejoindra automatiquement la session.

C'est normal que macOS vous demande de confirmer l'ouverture d'une application téléchargée depuis Internet.

4. Autoriser ISL Light Client

Configurez les autorisations nécessaires pour permettre à notre équipe d'accéder à votre écran.

Ouvrir les Préférences Système

Cliquez sur le logo Apple en haut à gauche, puis sélectionnez "Préférences Système" (ou "Réglages Système" sur les versions récentes).

Sécurité et Confidentialité

Accédez à l'onglet "Sécurité et confidentialité", puis sélectionnez "Confidentialité".

Activer l'Accessibilité

  1. Dans le menu de gauche, sélectionnez "Accessibilité"
  2. Cliquez sur le cadenas en bas à gauche de la fenêtre
  3. Entrez votre mot de passe administrateur
  4. Cochez la case à côté de "ISL Light Client"

Activer l'Enregistrement de l'écran

  1. Dans le même menu de gauche, sélectionnez "Enregistrement de l'écran"
  2. Cochez la case à côté de "ISL Light Client"

Important :

Une fois les autorisations activées, retournez sur le client ISL Light Client et cliquez sur "Arrêter", puis sur "Commencer" pour redémarrer la session avec les nouvelles autorisations.
